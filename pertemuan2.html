Operator dan Struktur Percabangan dalam C++

🎯 Tujuan Pembelajaran

  • Memahami konsep operator dalam C++.
  • Mampu menggunakan struktur percabangan (if, else if, else, switch) dalam program.
  • Menyelesaikan soal-soal sederhana menggunakan logika percabangan.

📚 Materi Pembelajaran

📝 2.1 Apa itu Operator?

  • Operator : Operator adalah simbol yang digunakan untuk melakukan operasi terhadap satu atau beberapa nilai (operand).

⚙️ 2.2 Jenis-Jenis Operator dalam C++

Operator digunakan untuk melakukan operasi terhadap variabel dan nilai dalam program.

📋 Tabel Jenis Operator:
Jenis Operator Contoh Keterangan
Operator Aritmatika +, -, *, /, % Untuk operasi matematika
Operator Perbandingan ==, !=, <, >, <=, >= Untuk membandingkan dua nilai
Operator Logika && Ampersand, Penyesuaian 2 kondisi

✏️ 2.3 Contoh Operator Aritmatika


#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;

    cout << "Penjumlahan: " << a + b << endl;
    cout << "Pengurangan: " << a - b << endl;
    cout << "Perkalian: " << a * b << endl;
    cout << "Pembagian: " << a / b << endl;
    cout << "Modulus: " << a % b << endl;

    return 0;
}
📊 Output:
Penjumlahan: 13
Pengurangan: 7
Perkalian: 30
Pembagian: 3
Modulus: 1

📝 2.4 Apa itu Struktur Percabangan?

  • Percabangan : Percabangan adalah proses memilih salah satu dari beberapa jalur eksekusi program berdasarkan kondisi tertentu.

✏️ 2.5 Percabangan if, else if, dan else

📘 Struktur Umum:

if (kondisi) {
    // kode jika kondisi benar
} else if (kondisi lain) {
    // kode jika kondisi lain benar
} else {
    // kode jika semua kondisi salah
}

✏️ 2.6 Contoh Program Percabangan Sederhana

📄 Contoh 1: Mengecek Bilangan Positif atau Negatif

#include <iostream>
using namespace std;

int main() {
    int angka;
    cout << "Masukkan angka: ";
    cin >> angka;

    if (angka > 0) {
        cout << "Angka positif." << endl;
    } else if (angka < 0) {
        cout << "Angka negatif." << endl;
    } else {
        cout << "Angka nol." << endl;
    }

    return 0;
}
🔍 Penjelasan:
  • Jika angka lebih besar dari 0 ➔ "positif"
  • Jika angka lebih kecil dari 0 ➔ "negatif"
  • Jika angka 0 ➔ "nol"

✏️ 2.7 Percabangan switch case

📘 Struktur Umum:

switch (variabel) {
    case nilai1:
        // kode
        break;
    case nilai2:
        // kode
        break;
    default:
        // kode jika tidak ada yang cocok
}

✏️ 2.8 Contoh Program switch case

📄 Contoh 2: Menentukan Hari

#include <iostream>
using namespace std;

int main() {
    int hari;
    cout << "Masukkan angka 1-5: ";
    cin >> hari;

    switch (hari) {
        case 1:
            cout << "Senin" << endl;
            break;
        case 2:
            cout << "Selasa" << endl;
            break;
        case 3:
            cout << "Rabu" << endl;
            break;
        case 4:
            cout << "Kamis" << endl;
            break;
        case 5:
            cout << "Jumat" << endl;
            break;
        default:
            cout << "Angka tidak valid" << endl;
    }

    return 0;
}
