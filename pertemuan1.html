ğŸ“ 1.1 Apa itu Logika & Algoritma?

  • Logika : Logika adalah ilmu berpikir sistematis, teratur, dan benar. Dalam pemrograman, logika digunakan untuk menyusun langkah penyelesaian masalah sehingga bisa diterjemahkan menjadi program.
  • Algoritma : Algoritma adalah urutan langkah-langkah logis untuk memecahkan masalah. Syarat algoritma yang baik:
    • Jelas : setiap langkah dapat dipahami.
    • Terbatas : algoritma harus berakhir, tidak berjalan terus menerus.
    • Efektif : dapat diselesaikan dengan langkah-langkah sederhana.

ğŸ“˜ Algoritma bisa disajikan dalam:
  • Pseudocode: Deskripsi langkah-langkah algoritma dalam bahasa sehari-hari yang mirip kode.
  • Flowchart: Diagram alur untuk menggambarkan langkah-langkah algoritma secara visual.

Contoh Program C++ (Dev-C++):

ğŸ” Penjelasan Program:
  • #include <iostream>: Untuk memasukkan input/output.
  • using namespace std;: Supaya bisa menggunakan cout dan cin tanpa perlu std::.
  • cin >>: Digunakan untuk menerima input dari user.
  • cout <<: Digunakan untuk menampilkan hasil ke layar.
  • Variabel bil_1, bil_2, dan hasil bertipe float untuk menerima angka desimal.

âœï¸ Tipe Data di C++ (Penjelasan Lengkap)

Dalam C++, tipe data digunakan untuk menentukan jenis nilai dari sebuah variabel.

ğŸ“‹ Tabel Tipe Data:
Tipe Data Deskripsi Contoh Penggunaan
int Bilangan bulat (tanpa desimal) int umur = 20;
float Bilangan pecahan (desimal) dengan ketelitian sedang float berat = 65.5;
double Bilangan pecahan (desimal) dengan ketelitian lebih tinggi dari float double pi = 3.14159;
char Menyimpan 1 karakter char grade = 'A';
string Menyimpan kumpulan karakter (teks) string nama = "Budi";
bool Tipe data logika: true atau false bool lulus = true;

ğŸ“œ Contoh Penggunaan Tipe Data:
      #include 
      using namespace std;
      
      int main() {
          int umur = 20;
          float berat = 55.5;
          double tinggi = 170.25;
          char inisial = 'B';
          string nama = "Budi";
          bool lulus = true;
      
          cout << "Nama: " << nama << endl;
          cout << "Inisial: " << inisial << endl;
          cout << "Umur: " << umur << " tahun" << endl;
          cout << "Berat: " << berat << " kg" << endl;
          cout << "Tinggi: " << tinggi << " cm" << endl;
          cout << "Status Lulus: " << lulus << endl;
      
          return 0;
      }

ğŸ“Š Output dari Program:
      Nama: Budi
      Inisial: B
      Umur: 20 tahun
      Berat: 55.5 kg
      Tinggi: 170.25 cm
      Status Lulus: 1

Catatan: Untuk tipe bool, true akan ditampilkan sebagai angka 1, dan false sebagai 0.
