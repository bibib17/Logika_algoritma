3.1 Apa itu Perulangan (Loop)?

Perulangan (loop) adalah proses eksekusi perintah atau instruksi yang dilakukan berulang-ulang berdasarkan suatu kondisi tertentu.

Perulangan sangat berguna untuk tugas yang dilakukan berulang tanpa harus menulis kode banyak kali.

3.2 Jenis-Jenis Perulangan dalam C++

Jenis Perulangan Penjelasan Singkat
for loop Digunakan untuk perulangan yang jumlahnya sudah diketahui.
while loop Digunakan untuk perulangan berdasarkan kondisi.
do while loop Sama seperti while, tapi perintah dijalankan minimal satu kali.

3.3 Struktur dan Contoh Perulangan

3.3.1 for Loop

Struktur Umum:

for (inisialisasi; kondisi; update) {
    // kode yang diulang
}

Contoh Program:

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << "Hello World ke-" << i << endl;
    }
    return 0;
}

Output:


Hello World ke-1
Hello World ke-2
Hello World ke-3
Hello World ke-4
Hello World ke-5
3.3.2 while Loop
while (kondisi) {
    // kode yang diulang
}

Contoh Program:

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 3) {
        cout << "Belajar C++" << endl;
        i++;
    }
    return 0;
}
3.3.3 do while Loop
do {
    // kode yang diulang
} while (kondisi);

Contoh Program:

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    do {
        cout << "Saya belajar perulangan" << endl;
        i++;
    } while (i <= 2);
    return 0;
}

3.4 Contoh Soal Sederhana Perulangan

Contoh 1: Menampilkan Angka 1 sampai 5
#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 5; i++) {
        cout << i << " ";
    }
    return 0;
}

Output:

1 2 3 4 5

Contoh 2: Menampilkan "Selamat Belajar" 3 kali
#include <iostream>
using namespace std;

int main() {
    int hitung = 0;
    while (hitung < 3) {
        cout << "Selamat Belajar" << endl;
        hitung++;
    }
    return 0;
}

Contoh 3: Angka Mundur dari 3 ke 1
#include <iostream>
using namespace std;

int main() {
    int angka = 3;
    do {
        cout << angka << endl;
        angka--;
    } while (angka > 0);
    return 0;
}
